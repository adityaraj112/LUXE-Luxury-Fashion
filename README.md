# 🛍️ LUXE – Luxury Fashion E-commerce Website

LUXE is a modern, responsive luxury fashion e-commerce platform designed to deliver a premium shopping experience. It features a clean UI, smooth navigation, and advanced frontend functionality built using HTML, CSS, and JavaScript.

---

## 🚀 Live Demo

🔗 https://adityaraj112.github.io/LUXE-Luxury-Fashion/

---

## ✨ Features

* 🛒 Add to Cart & Cart Management
* ❤️ Wishlist functionality
* 🌙 Dark Mode support
* 📄 Pagination system
* 🔍 Product listing & details page
* 🔐 Login page UI
* 💳 Checkout flow (UI-based)
* ⚡ Responsive design (mobile-friendly)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Modular CSS (base, layout, components)
* **Architecture:** Multi-page structure

```

```

2. Open the project folder

3. Run `index.html` in your browser
